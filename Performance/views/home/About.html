About

The Performance application demonstrates the implementation of the Serenity UI framework and widgets used to create a Single Page Application. The application also demonstrates how to create a custom Tile widget based on the serenity.Widget class.

Featured Serenity UI Widgets and Classes

  • Dropdownlist - The serenity.Dropdownlist widget is used in the Goal Edit page and the Achievement Edit page.
  • Model - The serenity.Model class is used for the Achievement, Goal, Lookup and Period models.
  • Notification - The serenity.Notification widget is used to display notifications when a save is successful or when there are validation errors.
  • Observable - The serenity.Observable class is used for all the View classes.
  • Router - The serenity.Router is used for client side routing.
